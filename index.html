Addendum:



1. html : The root element of an HTML document. It contains all other HTML elements.

2. head : Contains meta-information about the document, such as its title and links to scripts or stylesheets.

3. title : Specifies the title of the document, which is displayed in the browser's title bar or tab.

4. img : Incorporate in-line graphics (typically icons or small graphics) into an HTML document.

5. style : Contains CSS rules for styling HTML content. It's used here to set visual styles for various elements in the document.

6. div : A container tag that is used to group together and apply styles to other elements.

7. body : Contains the content of the document. Everything that is displayed in the browser window, apart from the title, is in the body.

8. p : Represents a paragraph of text.

9. h1 : Represents a level 1 heading. There are six levels of headings in HTML, from h1 (most important) to h6 (least important).

10. h2 : Represents a level 2 heading

11. footer: A structural element used to identify the footer of a page, document, article, or section.
